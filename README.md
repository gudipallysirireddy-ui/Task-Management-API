# Task Management API

## 🚀 Features
- User Registration & Login (JWT Authentication)
- Password Hashing (bcrypt)
- Create, Read, Update, Delete Tasks
- Protected Routes using JWT
- MongoDB Database Integration

## 🛠 Tech Stack
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT
- bcryptjs
